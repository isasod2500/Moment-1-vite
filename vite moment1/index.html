Moment 1: Automation, NPM och Vite

Skapa projektmapp med NPM och Vite som byggverktyg

Denna uppgift går ut på att skapa en webbplats med hjälp av standard HTML, CSS och Javascript med hjälp av automation av valfritt byggverktyg. I mitt fall valde jag Vite eftersom det kändes lättast att börja med ett byggverktyg redan central i kursen, och sedan leta mig fram till andra verktyg som passar mig mer.

Jag skapade projektmappen genom att skriva npm create vite@latest och välja inställningar så som Vanilla, Javascript och utan några experimentella funktioner.
